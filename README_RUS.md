# Lumina Tab

Lumina Tab — это чистое, настраиваемое и быстрое расширение для новой вкладки в Firefox, которое организует ваши закладки в сетку с предпросмотром папок.

## Установка

### 1. Временная

1. Откройте Firefox.
2. Перейдите по адресу `about:debugging#/runtime/this-firefox`.
3. Нажмите **«Загрузить временное дополнение…»**.
4. Выберите файл `manifest.json` из папки расширения.

### 2. Из Firefox Add-ons

Установите из официального магазина:  
👉 [Lumina Tab на Firefox Add-ons](тутбудетссылка.ру)
